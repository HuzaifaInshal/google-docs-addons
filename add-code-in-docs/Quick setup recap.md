## Quick setup recap (5 steps):

- Go to script.google.com → click + New project → name it "CodeBlocks Pro"
- In ⚙️ Project Settings → enable "Show appsscript.json in editor"
- Paste Code.gs content into the existing Code.gs tab, add a new HTML file named Sidebar and paste that content, then replace appsscript.json
- Click ▶ Run → onOpen → authorize when Google asks (click "Advanced → Go to app" — it's safe, it's your own script)
- Open any Google Doc → Extensions → CodeBlocks Pro → Open CodeBlocks Panel 🎉
